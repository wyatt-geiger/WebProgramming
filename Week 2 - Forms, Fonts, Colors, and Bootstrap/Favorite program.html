Party Invites Program

This program creates a party list and writes them to a text file using Python. It also validates whether or not the name already exists in the text file.

Here is some sample code from the beginning of the program:

import os
    invite_file = open('party_invites.txt','w')  # w for write mode             # change mode to a to build upon file instead of deleting it
    invite_file.write('Hostess - Betty Boop\n') # initialize a txt file
    invite_file.close()                                                         # initializing for homework
    # original input is a string, with commas separating each entry
    user_input = input('Enter guest names, separating each one with a comma. ') # rest of the program for homework
    user_input_list = user_input.split(',')  # change to a list
    user_input_fixed = ''  # a new string to hold the fixed items, so they can be written to the file
    for i in range(len(user_input_list)):
        guest_fixed = user_input_list[i].strip()
        user_input_fixed = 'Guest ' + str(i + 1) + ' - ' + guest_fixed + '\n'
        invite_file = open('party_invites.txt','a')  # 'a' for append mode
        invite_file.write(user_input_fixed)
    invite_file.close()
